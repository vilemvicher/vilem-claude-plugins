# Hello Command

Greet the user named "$ARGUMENTS" warmly and ask how you can help them today. Make the greeting personal and encouraging.
If not provided, greet without name.

Tell them what lines they have selected which are "$SELECTION" and tell them in which "$FILE" also.
If some of these are missing do not include them in the message about selection and file.
